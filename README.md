# Algorithm Study ICE

## **_Language_**

`JAVA`, `PYTHON`, `JavaScript`, `C++`

## **_Rule_**

- 매주 월요일 리뷰 진행.
- 화수목 -> `목요일 자정`까지 / 금토일 -> `일요일 자정`까지
- 매주 월, 목마다 번갈아가며 문제 선별해서 업로드
  - 화수목 (난이도 중하), 금토일 (난이도 중상)
- 초기에는 `백준`에서 문제에서 선별하기. (백준 레포 참고)

## **_Manual_**

**_파일명_**

- `알고리즘사이트_문제이름_이름`
  <br>

**_커밋 메시지_**

- `[complete] 알고리즘사이트_문제명`
- `[-ing] 알고리즘사이트_문제명`
